quotes-demo
===========

Description
-----------

- / = Frontend = Next.js
- /api = Backend = An API to store quotes (Django Rest Framework)
- /admin = Django Admin
- A scheduler to crawl quotes from http://quotes.toscrape.com (Airflow)

Tech Stack
----------

- Python
- Django Rest Framework
- uv
- Next.js
- Airflow

Usage
-----

```
git clone https://github.com/luc-phan/quotes-demo.git
cd quotes-demo
```
