quotes-demo
===========

Description
-----------

- /api = Backend = An API to store quotes (Django Rest Framework)
- / = Frontend = Next.js
- A scheduler to crawl quotes from http://quotes.toscrape.com (Airflow)

Tech Stack
----------

- Python
- Django Rest Framework
- uv
- Next.js
- Airflow

Usage
-----

```
git clone <repository-url>
cd quotes-demo
```
